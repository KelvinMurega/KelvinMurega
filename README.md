## Featured Projects

### 🌍 FloodScout — ML Geospatial Risk Scoring Engine
A machine learning-powered flood monitoring system for Kenya, combining geospatial data, a PostGIS database, and an interactive Leaflet.js map interface to score and visualise flood risk at a regional level.
`Python` `Django` `Scikit-Learn` `XGBoost` `PostgreSQL/PostGIS` `QGIS` `Leaflet.js`

### 🏛️ Public Environmental Grievance Redressal System
A web platform enabling citizens to log, track, and escalate environmental complaints to the relevant authorities, built to streamline accountability in public environmental governance.

### ☁️ Dynamics 365 Customer Service Enterprise Solutions
Enterprise CRM configuration and customisation work completed during an internship at Agile Cloud Limited, focused on customer service workflows and Microsoft Dynamics 365 deployments.
